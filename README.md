## Beyond Compare 5 能做什么？

简单说，就是把两个东西"摆在一起看差异"——但这个"东西"可以是：

- **文本 / 代码文件**：逐行高亮标注差异，改了哪里一目了然
- **文件夹 / 目录**：两个文件夹并排比较，新增、删除、修改的文件全部标出
- **Excel 表格**：支持多个工作表同时对比，财务和数据分析的福音
- **图片**：叠加、差值显示，轻松找出设计稿的细微改动
- **音频文件**：支持 MP3、MP4、AAC、FLAC 格式对比（这是 v5 新增的）
- **FTP / 云盘**：直接对比本地与远程服务器内容，不用先下载再比

到了第五版，还加入了很多实用升级：原生支持苹果 M 系芯片（Apple Silicon 用户终于不用跑 Rosetta 了）、深色模式、文本自动换行、右键上下文菜单……这些细节对日常使用体验影响挺大的。

## 哪些人最适合用 Beyond Compare？

**程序员**是最核心的用户群。代码 review、git diff 之外需要更直观的比较界面，BC 用了基本上就回不去了。很多人甚至把它配置成 git 的默认 diff 工具。

**法务/合规**也特别依赖这类工具。合同来回修改多个版本，用 Word 的"比较文档"功能凑合当然可以，但 Beyond Compare 的展示更清晰，还能批量处理文件夹。

**IT 运维**用它做服务器文件对比、配置文件同步，尤其是 SFTP 直连功能（专业版）能省很多手动操作的麻烦。

**设计师**核对切图版本、比较不同导出参数下的图片差异，BC 的图片对比功能（叠加显示、差异标红）比肉眼看靠谱多了。
